# Project VOLT | Multiverse  🚀

这是一个具有多重宇宙主题风格的个人博客网站，融合了赛博朋克、圣所流体神性和Windows 7复古风格等多种视觉体验。

## 网站特点 ✨

- **多主题切换** 🔄：包含四种截然不同的视觉主题，点击导航按钮可在不同"宇宙"间切换
  - 赛博朋克主题：霓虹色调、网格背景、故障动画效果 🔌
  - 圣所主题：简洁优雅的流体神性风格，柔和的色彩过渡 ⚪
  - Windows 7主题：模拟经典Windows 7界面，包含任务栏、窗口和桌面图标 💻
  - 错误主题：模拟系统崩溃的蓝屏效果，带来戏剧性体验 ⚠️

- **视觉特效** 🌈：每个主题都有独特的动画和交互效果，包括悬浮动画、扫描线、故障效果等

- **响应式设计** 📱：适配不同屏幕尺寸，在移动设备和桌面设备上均有良好表现

## 技术栈 🛠️

- HTML5 语义化结构 🏗️
- CSS3 用于样式和动画效果 🎨
- 原生JavaScript 用于交互逻辑（推测）📜

## 目录结构 📂

```
index.html - 主页面文件，包含所有主题和布局
```

## 使用方法 📝

1. 克隆或下载项目文件 📥
2. 直接在浏览器中打开`index.html`文件 🌐
3. 使用导航按钮切换不同主题页面 🔘

## 自定义指南 ⚙️

- 如需修改内容，可直接编辑`index.html`中的相应部分 📝
- 如需调整样式，可修改CSS中的变量和规则 🎨
- 新主题可通过添加CSS类和相应的JavaScript逻辑实现 🆕
